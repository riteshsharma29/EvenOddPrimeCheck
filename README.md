# EvenOddPrimeCheck
Python program to check if a number is Even / Odd  / Prime <br>

EvenOddPrimeCheck<br>
===================<br>

EvenOddPrimeCheck is simple Python library to test if a number is Even OR Odd OR Prime.<br>

----------<br>
Quick test<br>

```python<br>
import EvenOddPrimeCheck as ep<br>

ep.EOP_Checker(122)
ep.EOP_Checker(1)
ep.EOP_Checker(2)
ep.EOP_Checker(111)
ep.EOP_Checker(3)
```
